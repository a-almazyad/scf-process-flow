# SCF Process Flow

An interactive supply chain finance operating workflow covering:

- Supplier onboarding, including As-is and API-enabled To-be journeys
- Invoice financing and disbursement
- Settlement and loan closure

## Live site

[Open the published workflow](https://a-almazyad.github.io/scf-process-flow/)

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The project uses Vite's single-file build so the generated workflow can also be opened as a portable HTML file.

To rebuild the GitHub Pages artifact:

```bash
npm run build:pages
```
